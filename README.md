# Customer-churn-analysis
Customer churn analysis using Excel

## Project Overview
This project analyzes customer churn data to identity patterns and factors contributing to customer exit.
The dashboard provides a visual summarry of customer behaviour, trends and key metrics.

## Key Insights
- Higher churn observe among short-tenure customers
- Inactive customers show higher exit rates

## Recommendation
- Improve onboarding for new customers
- Increase engagement for inactive customers
